# Color Palette

This document contains the color definitions used in the Frigo Express BG website.

## Primary Colors

### Blue
- **Hex**: `#25425E`
- **Usage**: Primary brand color, used for headings, buttons, and key UI elements
- **Examples**: 
  - Text: `text-[#25425E]`
  - Background: `bg-[#25425E]`
  - Border: `border-[#25425E]`

### Green
- **Hex**: `#C8E6C9`
- **Usage**: Accent color, used for highlights and buttons
- **Examples**:
  - Text: `text-[#C8E6C9]`
  - Background: `bg-[#C8E6C9]`

## Tailwind Config Colors

These colors are defined in `tailwind.config.js` and can be used with Tailwind classes:

- `primary-evergreen`: `#61644F`
- `primary-deepsea`: `#3F4D59`
- `primary-sand`: `#EAE5E1`
- `primary-dune`: `#CFC5B8`
- `accent-terracotta`: `#965939`
- And more...

## Notes

- The blue color (`#25425E`) is currently used as inline hex values throughout the codebase
- Consider adding these colors to `tailwind.config.js` for easier management
- The green color (`#C8E6C9`) is also used as inline hex values
